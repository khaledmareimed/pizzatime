# Security Instruction

## Function Organization
- If you want to use a function, make it in a folder called `funcs` and then use it
- Don't add functions directly to the UI components

## Security Priority
- Make sure to make security number 1 priority in all development decisions

## UI Guidelines
- Make sure the UI supports both dark and light themes depending on the mobile settings
- Make sure the UI maintains consistency with the overall app UI design and patterns
